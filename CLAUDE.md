# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## Overview

Bun monorepo for Obsidian.md customizations. Packages under `packages/`:

- **templater-functions** — TypeScript user scripts for the Templater plugin, built to `dist/templater/`
- **quickadd-scripts** — TypeScript user scripts for the QuickAdd plugin, built to `dist/quickadd/`
- **css-snippets** — CSS customizations copied to `dist/snippets/`
- **web-clipper** — Web Clipper templates defined in PKL, generating JSON to `dist/web-clipper/`
- **raycast-scripts** — Raycast script commands that invoke Obsidian via the `obsidian://` URI scheme

## Commands

### Repo root

```sh
bun run fmt    # format with oxfmt
bun run lint   # lint with oxlint
```

Each package also exposes `fmt` / `lint` scripts (`bun --cwd packages/<name> run fmt|lint`).

### templater-functions / quickadd-scripts

```sh
bun run build   # compile TypeScript → dist/{templater,quickadd}/ (CommonJS, minified)
```

### css-snippets

```sh
bun run build   # copy ./*.css → dist/snippets/
```

### web-clipper

```sh
bun run eval   # pkl eval → generate dist/web-clipper/*.json from *.pkl
bun run fmt    # pkl format
```

## Architecture

### Package relationships

`dist/web-clipper/default.json` references `unique_note()` from `templater-functions`, linking the two packages at the Obsidian layer (not at the code level).

### templater-functions / quickadd-scripts

- Sources live at the package root (`*.ts`); `build.ts` lists entries explicitly
- Bun bundler emits CommonJS targeting Node
- `templater.d.ts` / `quickadd.d.ts` declare the global `Tp` / `Qa` types injected by each plugin
- User scripts must use `module.exports = fn` (not `export default`) — both Templater and QuickAdd require `module.exports` to be the function directly

### web-clipper

- Templates: `*.pkl` — each file `amends "@obsidian.webClipper/Template.pkl"` and sets template-specific values
- Schema dependency: `obsidian.webClipper` package defined in `PklProject`
- Output: `dist/web-clipper/*.json` — generated by `bun run eval`, committed to the repo
- Property values containing `\"` use PKL raw strings (`#"..."#` or `#"""..."""#`)

### raycast-scripts

- Bash scripts with Raycast `@raycast.*` headers; no build step
- Invoke Obsidian via `obsidian vault=vault quickadd choice=... value-<var>=...` (QuickAdd URI handler only reads params prefixed with `value-`)

### Tooling

- **Runtime**: Bun (not Node.js)
- **Formatting**: oxfmt (Rust-based); `dist` is ignored
- **Linting**: oxlint; `dist` is ignored
- **Template generation**: PKL (`pkl eval`)
- Shared dependency versions are pinned via Bun workspace catalog in the root `package.json`
